# Data Formats

The following lists supported Data Formats

* [Base64](http://camel.apache.org/base64.html)
* [Bindy](http://camel.apache.org/bindy.html)
* [Castor](http://camel.apache.org/castor.html)
* [Crypto](http://camel.apache.org/crypto.html)
* [CSV](http://camel.apache.org/csv.html)
* [Flatpack](http://camel.apache.org/flatpack.html)
* [GZip](http://camel.apache.org/gzip-data-format.html)
* [HL7](http://camel.apache.org/hl7.html)
* [JSON](http://camel.apache.org/json.html)
* [JAXB](http://camel.apache.org/jaxb.html)
* [Protobuf](http://camel.apache.org/protobuf.html)
* [SOAP](http://camel.apache.org/soap.html)
* [Serialization](http://camel.apache.org/serialization.html)
* [TidyMarkup](http://camel.apache.org/tidymarkup.html)
* [XmlBeans](http://camel.apache.org/xmlbeans.html)
* [XMLSecurity](http://camel.apache.org/xml-security-component.html)
* [XStream](http://camel.apache.org/xstream.html)
* [YAML](http://camel.apache.org/yaml-data-format.html)
* [Zip](http://camel.apache.org/zip-file-dataformat.html)
