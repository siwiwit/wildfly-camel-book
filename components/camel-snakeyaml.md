## camel-snakeyaml

The [camel-sankeyaml](http://camel.apache.org/yaml-data-format.html) component provides a Data Format to marshal and unmarshal Java objects to and from [YAML](http://www.yaml.org/). For YAML to object marshalling, Camel provides integration with the [SnakeYAML](http://www.snakeyaml.org/) library.
